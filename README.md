# 业鉴

> 业主的目光如镜，照见社区治理的每一个角落。

**业鉴**是一款面向住宅小区业主的微信小程序，核心使命是**业主赋权、透明监督**——让每一位业主都能便捷地参与小区治理、监督业委会和物业公司，让社区事务在阳光下运行。

---

## ✨ 核心功能

### 🗳️ 合规投票
- 支持「双过半」「双¾」等法定表决规则
- 按建筑面积加权计票，自动判定表决结果
- 区块链存证（TBaaS），投票结果不可篡改
- 投票资格与身份核验等级联动

### 🔧 智能报修
- 一键提交报修工单，支持语音描述、拍照上传
- SLA 服务等级监控，超时自动预警升级
- 工单全流程状态追踪：待派单 → 处理中 → 待验收 → 已完成
- 维修完成后业主评价验收

### 💰 财务透明
- 物业费用收支明细公开，每一笔钱可追溯
- 财务凭证在线预览，支持区块链存证验证
- 审批联署流程可视化，业委会支出业主监督
- 缴费账单一目了然，微信支付 JSAPI 接口预留

### 🔐 四级身份核验
| 等级 | 认证方式 | 权限 |
|------|---------|------|
| L1 | 手机号验证 | 基础浏览 |
| L2 | 房产信息验证 | 投票、报修 |
| L3 | 人脸识别 | 发起投票、联署 |
| L4 | 线下核验 | 业委会/物业端管理 |

### 📊 物业管理看板（物业端）
- 工单派单与 SLA 实时监控
- 超时预警与自动升级
- 数据统计与趋势分析

## 🎨 设计语言

基于 [Stripe Design Language](https://stripe.com)，追求金融级信任感与克制精致：

- **品牌色**：Stripe 紫 `#533afd` — 主交互、CTA、选中态
- **头部渐变**：深靛蓝 `#1c1e54 → #2a2d7a → #3d40a8`
- **阴影系统**：蓝色调多层阴影，品牌感深度
- **图标体系**：43 个 Stripe 风格线条图标，1.5px 描边，base64 内联
- **字重**：标题 300-500（Stripe 轻量签名风），正文 400
- **圆角**：保守克制，4-16rpx

完整设计规范见 [DESIGN.md](./DESIGN.md)。

## 🚀 快速开始

### 前置条件
- [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 微信小程序 AppID（可在[微信公众平台](https://mp.weixin.qq.com/)注册）

### 运行项目

1. **克隆仓库**
   ```bash
   git clone https://github.com/Sandy-di/tr-owner-miniapp.git
   ```

2. **导入项目**
   - 打开微信开发者工具 → 导入项目
   - 选择项目目录
   - 填入 AppID（测试可使用测试号）

3. **预览运行**
   - 项目内置 Mock 数据，导入后即可预览全部功能
   - 无需额外配置后端服务

### 对接真实后端

项目当前使用 `utils/mock.js` 提供模拟数据。对接真实后端时：

1. 在 `app.js` 的 `globalData.baseUrl` 中配置 API 地址
2. 将各页面中调用 `mock.js` 的方法替换为 `wx.request()` 调用
3. 移除 `utils/mock.js` 的引用
